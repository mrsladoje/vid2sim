You are an award-winning frontend engineer and creative developer.

Your task is to design and build a modern, clean, award-winning website UI for a product called "Vid2Sim".

## PRODUCT IDEA
Vid2Sim is a tool that transforms real-world video (captured using an OAK-4 camera) into an interactive, in-browser physics simulation. Users upload a video, and the system reconstructs objects that can be manipulated in a 3D environment.

IMPORTANT:
- This task is FRONTEND ONLY (no backend or AI implementation)
- Focus on UX, UI, transitions, and 3D presentation
- Fake/mock the processing step where needed

Match the style of premium creative developer websites:
- Minimal but futuristic (inspired by Apple, Linear, and Awwwards-winning sites)
- Dark theme with subtle gradients and glow
